# Motus

An independent, open-source iOS groovebox / grid-sequencer app with AUv3 plugin
support, for iPhone and iPad.

## Status

Open source under the [MIT License](LICENSE). Not on the App Store — build and
run it yourself from source.

## Building

Open the project in Xcode and build to your device. Signing uses your own Apple
Developer team; set `DEVELOPMENT_TEAM` in `project.yml` / the Xcode project to
your team ID.
